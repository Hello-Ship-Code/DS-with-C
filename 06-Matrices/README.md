## Diagonal Matrix

A Matrix `m` `n x n` is said to be Diagonal Matrix. if all the values of the matrix are `0` except the diagonal elements.
$$m[i,j]=0  if(i==j) $$
m =
| 1 0 0 0 | <br/>
| 0 2 0 0 | <br/>
| 0 0 3 0 | <br/>
| 0 0 0 4 | <br/>
